==================================================
  S&P 500 Watchlist - 사용방법
==================================================

[ 필수 설치 프로그램 - 처음 한 번만 ]
--------------------------------------------------
1. Python 3.10 이상
   https://www.python.org/downloads/
   ※ 설치 시 "Add Python to PATH" 체크 필수

2. Node.js 18 이상
   https://nodejs.org/


[ 실행 방법 ]
--------------------------------------------------
1. 이 폴더(S-P500-main)에서 "start.bat" 더블클릭

2. 창이 2개 열립니다:
   - S&P500 Backend  : 백엔드 서버 (FastAPI)
   - S&P500 Frontend : 프론트엔드 서버 (Vite)

3. 백엔드 창에 아래 메시지가 뜨면 준비 완료:
   Uvicorn running on http://0.0.0.0:8000

4. 브라우저에서 접속:
   http://localhost:5173


[ 최초 실행 시 주의사항 ]
--------------------------------------------------
- 처음 실행하면 pip install(패키지 설치)이 자동으로 됩니다.
  1~3분 정도 소요되니 백엔드 창이 완료될 때까지 기다리세요.

- 서버가 뜬 직후에는 S&P 500 종목 데이터를 자동 수집합니다.
  이 작업은 2~5분 걸리며, 그 동안 목록이 비어 보일 수 있습니다.
  잠시 후 새로고침하면 데이터가 채워집니다.

- 두 번째 실행부터는 빠르게 시작됩니다.


[ 종료 방법 ]
--------------------------------------------------
백엔드/프론트엔드 창을 각각 닫거나,
창 안에서 Ctrl+C 를 누르면 종료됩니다.


[ 오류 발생 시 ]
--------------------------------------------------
- "포트 이미 사용 중" 오류가 뜨면:
  이전에 실행한 창이 아직 열려 있는지 확인하고 닫은 후 재실행

- 백엔드 창이 바로 닫히면:
  창 안의 에러 메시지를 확인 후 개발자에게 문의


[ 접속 주소 요약 ]
--------------------------------------------------
앱 화면    : http://localhost:5173
API 문서   : http://localhost:8000/docs
헬스체크   : http://localhost:8000/api/health
